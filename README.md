# liberMobile
Libertas per Litteras
## The project
liberMobile started as a project to create a mobile application for the [LiberLiber](https://www.liberliber.it) website.  
The project is currently in the early stages of development and is not yet ready for production use. 
### Objectives
The main objectives of the project are:
- Loading and displaying the books from PDF, EPUB, ODT and other formats downloaded from the LiberLiber website
- Making a good audiobook player
- Making it possible to sync audiobooks and ebooks chapters, making it possible to read and listen to the same book at the same time
- Starting the project for iOS
- Add translations for multiple languages
- Add compatibility with multiple sources across the web
- Creating a website for the project
